# homebrew-mini-agent

Homebrew tap for [mini-agent](https://github.com/kowyo/mini-agent) - A minimal agent running in your terminal.

## Installation

```bash
# Add the tap
brew tap kowyo/mini-agent

# Install mini-agent
brew install mini-agent
```

Or in one command:

```bash
brew install kowyo/mini-agent/mini-agent
```

## Requirements

- Python 3.14 (installed automatically via Homebrew)
- macOS or Linux

## Usage

After installation, run:

```bash
mini-agent --version
mini-agent --help
```

## Updating

```bash
brew update
brew upgrade mini-agent
```

## Uninstalling

```bash
brew uninstall mini-agent
brew untap kowyo/mini-agent
```
