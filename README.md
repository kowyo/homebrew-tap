# homebrew-tap

A personal Homebrew tap for Kowyo's projects.

## Installation

```bash
brew tap kowyo/tap
```

## Available Formulae

| Formula | Description | Install |
|---------|-------------|---------|
| mini-agent | A minimal agent running in your terminal | `brew install mini-agent` |

## Usage

Install any formula with:

```bash
brew install kowyo/tap/<formula>
```

Or tap first, then install:

```bash
brew tap kowyo/tap
brew install <formula>
```
